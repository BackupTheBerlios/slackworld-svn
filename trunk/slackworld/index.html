Introduction

Welcome to the Slack World website! The Slack World is a monthly Slackware Linux magazine. The main aim of the project is to fill a few gaps that we see in covering Slackware-related information by other resources, by providing in-depth articles and a summary of recent Slackware-related news.

We are currently working on the first issue, please stay tuned!

Contributing

We are in a constant look-out for skilled authors. If you think you can contribute to The Slack World, don't hesitate to contact us.
